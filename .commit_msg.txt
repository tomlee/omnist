feat: add omnist CLI scaffolding with format and schema format commands

PR2 of the CLI implementation arc (docs/design/cli-spec.md). Adds
omnist/cli.py: argparse-based dispatch, shared stdin/stdout/file I/O
helpers, and two fully-implemented commands:

- omnist format <input> [-o OUTPUT] -- canonicalize OML (read_oml ->
  write_oml).
- omnist schema format <schema-file> [-o OUTPUT] -- canonicalize OSD
  (parse_schema -> to_dsl), safe reformat only.

No new dependency; [project.scripts] entry installs the omnist command.
Malformed input raises the library's own ParseError/SchemaError, printed
cleanly to stderr with exit code 2, never an uncaught traceback.

Adds docs/cli.md, wires it into mkdocs nav/docs index/README, version
bump to 0.2.3, CHANGELOG entry.
